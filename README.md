# 🛒 Blinkit Sales Dashboard (Power BI)
## 📌 Project Overview
This project is a **Blinkit Sales Dashboard** built using Power BI to analyze sales performance across multiple dimensions such as outlet size, location, item type, and customer preferences.
The dashboard transforms raw data into **interactive insights**, helping in better and faster business decision-making.

## 🎯 Problem Statement
To analyze Blinkit's sales data and identify key factors affecting performance, including:
* Outlet size and type
* Product categories (fat content)
* Outlet location
* Establishment trends
* Customer satisfaction (ratings)
  
## 🛠️ Tech Stack
* **Power BI** → Data visualization & dashboard creation
* **Microsoft Excel** → Data cleaning & preprocessing
* **SQL** → Data filtering and query-based analysis

## 📊 Key Features / KPIs
* 💰 **Total Sales**
* 🏬 **Sales by Outlet Size**
* 🏪 **Sales by Outlet Type**
* 🥗 **Sales by Fat Content**
* 📍 **Sales by Outlet Location**
* 📅 **Outlet Establishment Trends**
* 📈 **Average Sales**
* ⭐ **Average Rating**

## 📈 Data Visualization Techniques
**The dashboard uses multiple visual elements to enhance data understanding:**
* 📊 **Bar Chart → Comparison of sales across categories**
* 📉 **Column Chart → Trend analysis over time**
* 🍩 **Donut Chart → Proportion of sales by segments**
* 🥧 **Pie Chart → Distribution insights**
* 🔻 **Funnel Chart → Stage-wise data breakdown**
* 📍 **Scatter Plot → Relationship between variables (e.g., sales vs rating)**
* 🎛️ **Slicers → Interactive filtering for dynamic analysis**

## 🔍 Data Processing Steps
1. Cleaned raw data using Excel
2. Removed duplicates and handled missing values
3. Structured dataset for analysis
4. Applied SQL queries for filtering and segmentation
5. Built data model in Power BI
6. Created interactive visuals and dashboard

## 📌 Insights Gained
* Identified top-performing outlet sizes and types
* Analyzed customer preferences based on fat content
* Compared location-wise sales performance
* Observed trends based on establishment year
* Evaluated customer satisfaction using ratings

## 🚀 Impact
* ⏱️ **Reduced manual analysis time by ~60–70% through dashboard automation**
* 📊 **Improved decision-making speed using interactive filters and slicers**
* 📌 **Enabled faster identification of high-performing outlets and product categories**
